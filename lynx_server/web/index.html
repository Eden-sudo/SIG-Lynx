Presentación del Proyecto

SIG-Lynx es un sistema de inspección geométrica, multicontrol con estructura en ROS2, permitiendo el control de diferentes dispositivos y permitiendo a diferentes máquinas con diversos ángulos de movimiento ser más precisos.

En esta demostración, exhibiremos el funcionamiento del sistema en un hardware de bajo costo; Lynxmotion, un sistema didáctico, para volverlo más eficiente y preciso.

Ingeniería de Precisión

SIG-Lynx garantiza la máxima precisión robótica mediante actuadores de alta respuesta acoplados a una arquitectura de cálculo inverso en tiempo real.

Visión Artificial

Identificación de patrones y extracciones vectoriales para la toma de decisiones críticas en trayectorias espaciales.
